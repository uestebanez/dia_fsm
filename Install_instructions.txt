Instructions to install DIA and FSM plug-in,these steps must be followed in
order:
	1. Execute python v2.3 installer 
	   (https://www.python.org/download/releases/2.3/)
	2. Execute DIA installer (https://sourceforge.net/projects/dia-installer/)
		(During DIA installation, make sure that "Python plug-in" checkbox is selected!)
	3. Execute pycairo installer
	(https://ftp.gnome.org/pub/GNOME/binaries/win32/pycairo/1.0/)
	4. Execute pygtk installer
	(https://ftp.gnome.org/pub/GNOME/binaries/win32/pygtk/2.8/)
	5. Execute copy_plugin_files.bat

Please,help us to improve this if you find problems or bugs!!!
